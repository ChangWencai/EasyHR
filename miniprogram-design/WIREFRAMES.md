### 组件说明

| 区域 | 组件 | 说明 |
|------|------|------|
| 顶栏 | NavBar | 左侧Logo + 右侧通知+设置图标 |
| 问候 | GreetingCard | 根据时间段显示"早/午/晚安"，显示部门+日期 |
| 数据卡片 | DataCard | 两列布局，本月工资+社保状态，tap 跳对应Tab |
| 公告列表 | NoticeList | 最多显示3条，更多跳公告列表页 |
| 快捷入口 | QuickEntryGrid | 4列网格，图标+文字，tap 跳对应功能页 |

### 跳转关系

```
首页快捷入口 → 合同列表 / 报销申请 / 我的 / 帮助页
通知图标    → 通知列表
工资卡片   → 工资条Tab（scroll到当月）
社保卡片   → 社保Tab
公告项    → 公告详情页
```
