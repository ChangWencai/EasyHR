# EasyHR 小程序色彩系统 — 视觉参考

## 色彩令牌一览

```
背景 & 表面
─────────────────────────────────────────
 bg          #F8FAFC   页面背景（冷灰白）
 surface     #FFFFFF   卡片/组件表面
 surface-2   #F1F5F9   次级表面（交替行/标签）

文本
─────────────────────────────────────────
 text-primary #0F172A   主标题/重要文字
 text-body    #334155   正文
 text-muted   #64748B   次要文字/占位符
 text-disabled #CBD5E1  禁用状态

品牌色
─────────────────────────────────────────
 primary     #4F46E5   主色（Indigo-600）
 primary-lt  #EEF2FF   浅色（Indigo-50，背景高亮）
 secondary   #7C3AED   紫（Violet-600）

功能色
─────────────────────────────────────────
 success     #10B971   绿色：成功/已缴
 warning     #F59E0B   橙色：待处理/提醒
 danger      #EF4444   红色：错误/删除
 info        #0EA5E9   蓝色：信息

边框 & 分隔
─────────────────────────────────────────
 border      #E2E8F0   默认边框
 border-focus #4F46E5  聚焦边框
```

## 应用场景示例

### 卡片设计
```
┌─────────────────────────────────────┐
│  Surface #FFFFFF                     │
│                                      │
│  标题文字 Primary #0F172A            │
│  正文文字 Body   #334155             │
│  辅助文字 Muted  #64748B             │
│                                      │
│  ✓ 已发放  (Success绿底 #10B971)    │
│  ⚠ 待发放  (Warning橙底 #F59E0B)    │
│                                      │
└─────────────────────────────────────┘
页面背景 bg #F8FAFC
卡片阴影 shadow-md: rgba(79,70,229,0.08)
```

### 按钮层级

```
主按钮: 背景 #4F46E5  文字 #FFFFFF
次按钮: 背景 #FFFFFF  边框 #4F46E5  文字 #4F46E5
幽灵按钮: 背景透明  文字 #4F46E5
危险按钮: 背景 #EF4444  文字 #FFFFFF
```

### TabBar 状态

```
未选中: 图标 #64748B  文字 #64748B
选中:   图标 #4F46E5  文字 #4F46E5  底部指示线 #4F46E5
```
