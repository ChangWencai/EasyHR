2. 在 `internal/employee/repository.go` 中新增方法:

   a. ListRoster(orgID int64, params ListQueryParams, page, pageSize int) ([]Employee, int64, error):
      - 在现有 List 基础上扩展 department_id 筛选
      - 如果 params.DepartmentID != nil, 添加 WHERE department_id = ?
      - 返回 Employee 切片 + total

   b. GetSalaryAmounts(orgID int64, employeeIDs []int64) (map[int64]float64, error):
      - 查询 salary_items 表，取每个 employee_id 最近的 effective_month 中 template_item 对应"基本工资"的 amount
      - 返回 employee_id -> amount 映射
      - 如果 salary_items 表不存在（HasTable 检查），返回空 map

   c. GetContractExpiryDays(orgID int64, employeeIDs []int64) (map[int64]*int, error):
      - 查询 contracts 表，取每个 employee_id 最近的 active 状态合同的 end_date
      - 计算 days = int(time.Until(end_date).Hours() / 24)
      - end_date 为 nil (无固定期限): map value = nil
      - 无合同记录: map 中无此 key
      - 如果 contracts 表不存在，返回空 map

   d. GetDepartmentNames(orgID int64, deptIDs []int64) (map[int64]string, error):
      - 查询 departments 表，返回 id -> name 映射
      - 如果 departments 表不存在，返回空 map
