## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline badge spacing |
| sm | 8px | Compact element spacing, header actions gap |
| md | 16px | Default element spacing, section margin-bottom, pagination top margin |
| lg | 20px | Section padding, page-level padding |
| xl | 24px | Page padding (home-view), major layout gaps |
| 2xl | 32px | Loading padding, drawer padding |
| 3xl | 48px | Major section breaks |

Exceptions:
- shortcut-icon 48x48px -- 图标容器固定尺寸

**Source:** 从 HomeView.vue 和 EmployeeList.vue 现有样式提取。全部为 4 的倍数。

## Interaction Patterns

### 花名册合同到期天数

- 到期天数 > 0: 普通文字显示 "{n}天"
- 到期天数 <= 0: 红色文字显示 "已过期{n}天"
- 无合同: 灰色文字 "无合同"
- 无固定期限合同: 显示 "无固定期限"

### 离职审批操作

- 同意: el-popconfirm 确认后调用 approve API，成功后刷新列表
- 驳回: 弹出 el-dialog 输入驳回原因（选填），确认后调用 reject API
- 去减员: router.push 携带 { employee_id, employee_name } 跳转社保减员页面
- 完成离职: el-popconfirm 确认后调用 complete API

### 组织架构搜索

- 输入关键字后 300ms 防抖搜索
- 匹配的节点高亮（蓝色 #1677FF），未匹配节点降低透明度 (opacity: 0.3)
- 清空搜索框恢复全部节点显示

### 员工信息登记转发

- 二维码弹窗: 展示链接对应的 QR Code（前端生成），可长按保存
- 复制链接: 使用 navigator.clipboard.writeText，成功后 ElMessage.success
- 短信发送: 调用后端 SMS API，发送后 ElMessage.success 显示手机号

### 花名册搜索增强

- 现有搜索支持姓名/手机号/岗位，新增部门筛选（el-select 下拉选择部门列表）
- 搜索参数增加 department_id
