## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| 员工浏览器 -> /registrations/:token/submit | 公开接口，无认证，仅 Token 验证 |
| 管理员 -> /registrations CRUD | 需要 JWT + OWNER/ADMIN 权限 |
| Registration 提交 -> Employee 更新 | 敏感数据写入，需加密 |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-05-05 | Tampering | Registration Token 暴力猜测 | mitigate | crypto/rand 32-byte (2^256 空间), 7天过期, Token 使用后标记为 used |
| T-05-06 | Spoofing | 恶意提交他人登记信息 | mitigate | Token 绑定 org_id + EmployeeID, 提交后 Token 失效 |
| T-05-07 | Information Disclosure | 公开接口返回敏感数据 | mitigate | GetRegistrationDetail 仅返回姓名/部门/岗位等基础信息，不返回手机号/身份证 |
| T-05-08 | Tampering | XSS 通过员工填写内容 | accept | Vue 模板自动转义，无富文本场景 |
| T-05-09 | Information Disclosure | 敏感字段存储 | mitigate | 手机号/身份证/银行卡使用 AES-256-GCM 加密 + SHA-256 哈希索引 |
</threat_model>
