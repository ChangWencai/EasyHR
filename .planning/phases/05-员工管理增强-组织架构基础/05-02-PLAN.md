   - CRUD 方法: CreateDepartment, UpdateDepartment, DeleteDepartment, ListDepartments

     a. ListAll 获取全部部门
     b. 通过 empRepo 查询全部在职员工（需新增 ListAllByOrg 方法）
     c. BuildTree 内存递归构建3层树:
        - 按 parent_id 分组部门
        - 根节点 = parent_id IS NULL 的部门
        - 递归构建子部门节点
        - 员工按 department_id 挂载到部门下，Type="employee"
        - 每个员工节点 Name = employee.Name, 但为了体现层级：部门->岗位(position)->员工
        - 中间层: 同一部门下按 position 分组创建岗位节点(Type="position"), 员工挂在岗位下
   - SearchTree(orgID int64, keyword string) 方法:
     a. 获取完整树
     b. 递归搜索匹配节点（部门名/岗位名/员工姓名 包含 keyword，大小写不敏感）
     c. 标记匹配节点: itemStyle.color = "#1677FF", label.color = "#1677FF", label.fontWeight = 700

```typescript
import request from './request'

export interface Department {
  id: number
  name: string
  parent_id: number | null
  sort_order: number
}

export interface TreeNode {
  id: number
  name: string
  type: 'department' | 'position' | 'employee'
  children?: TreeNode[]
}

export const departmentApi = {
  list: () => request.get<Department[]>('/departments'),
  getTree: () => request.get<TreeNode[]>('/departments/tree'),
  create: (data: { name: string; parent_id?: number | null; sort_order?: number }) =>
    request.post<Department>('/departments', data),
  update: (id: number, data: Partial<Department>) =>
    request.put<Department>(`/departments/${id}`, data),
  delete: (id: number) => request.delete(`/departments/${id}`),
}
```
