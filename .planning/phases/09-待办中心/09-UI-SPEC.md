## Spacing Scale

All values are multiples of 4px (8-point grid base).

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding |
| sm | 8px | Default card radius, compact element spacing |
| md | 16px | Default element spacing, card padding |
| lg | 24px | Section padding, layout gaps |
| xl | 32px | Major section breaks |
| 2xl | 48px | Page-level spacing |

**Exceptions:**
- Touch target minimum: 44px (icon-only buttons, mobile)
- Card border-radius: `--radius-lg` = 12px (from variables.scss, consistent with RegisterPage.vue)
- Small element radius: `--radius-md` = 8px (todo-card, shortcut-item)

## Component Inventory

### 1. RingChartCard (HomeView.vue — new)
- **Purpose:** Show completion rate for all todos and time-limited tasks
- **States:** Loading (spinner), Empty (0 items, show "暂无数据"), Populated
- **Behavior:** On click nothing (read-only display)

### 2. TodoList (HomeView.vue + new TodoList page)
- **Purpose:** Full todo list with search, filter, sort, pin, export
- **Item states:**
  - `pending`: Normal card, `--text-primary` title
  - `completed`: Checkmark icon, `--success` text tint, strikethrough optional
  - `overdue`: Red badge ("超时"), `--danger` border-left accent
  - `expired`: Gray badge ("失效"), `--text-tertiary` muted
  - `terminated`: Gray badge ("已终止"), `--text-tertiary` muted, strikethrough title
- **Urgency badge rules (D-09-04):**
  - Normal: no badge
  - 1-7 days before deadline: yellow "即将到期" badge
  - Overdue 1-7 days: red "超时" badge, `--danger`
  - Overdue 8-14 days: red "超时" badge (same as above)
  - Overdue 15+ days: gray "失效" badge
- **Columns:** 名称 | 发起人 | 创建时间 | 截止日期 | 状态+徽章 | 限时类型 | 操作

### 3. TodoCard (HomeView.vue inline)
- **Extends existing:** Add urgency badge + time-limit indicator
- **Width:** `grid-template-columns: repeat(4, 1fr)` desktop, 3 tablet, 2 small tablet, 1 mobile
- **States:** pending (default), overdue (red left border), expired (gray left border)

### 4. CarouselBanner (HomeView.vue — new)
- **Purpose:** 1-3 images auto-cycling at top of HomeView
- **Behavior:** Auto-advance every 4 seconds, fade transition (confirmed: fade)
- **Indicators:** Dot indicators below, active dot uses `--primary`
- **Tap/click:** Navigate to configured URL
- **OSS URL:** `image_url` from `CarouselItem` table
- **Time window:** Hidden if `active=false` per asynq scheduler (D-09-07)

### 5. ShortcutGrid (HomeView.vue — existing + 3 new)
- **Existing 6:** 员工管理 (User) / 薪资管理 (Money) / 社保管理 (Umbrella) / 个税申报 (Document) / 凭证管理 (Wallet) / 发票管理 (Document)
- **New 3:**
  - 新入职 → `/employee/create` — UserAdd icon, bg `#E6F4FF`, color `#1677ff`
  - 调薪 → `/tool/salary` — TrendCharts icon, bg `#F6FFED`, color `#52c41a`
  - 考勤 → `/attendance/clock-live` — Clock icon, bg `#F9F0FF`, color `#722ed1`
- **Grid:** `grid-template-columns: repeat(9, 1fr)` desktop; responsive to 6→3→2 at same breakpoints as existing shortcuts

### 6. TodoItem (TodoRow in full list page)
- **Fields:** title / creator / created_at / deadline / status badge / is_time_limited badge / pinned / actions
- **Pin icon:** `Star` (Element Plus) filled = pinned, outline = unpinned
- **Actions dropdown (el-dropdown):** 邀请协办 | 终止任务 | 删除(only completed)
- **Status badge colors:**
  - 待办: `--warning: #FFAB00` with light bg
  - 已完成: `--success: #36B37E` with light bg
  - 超时: `--danger: #FF5630` with light bg
  - 失效: `#8C8C8C` with light bg
  - 已终止: `#8C8C8C` with light bg

### 7. CarouselManagePage (new page)
- **Purpose:** Admin configures 1-3 carousel images
- **Table columns:** 排序 | 预览图 | 跳转链接 | 生效时间 | 失效时间 | 状态 | 操作
- **Upload:** el-upload to OSS, max 3 images, image/* accept
- **Time range:** el-date-picker range, start_at / end_at
- **Link URL:** el-input for external or internal route
- **Active toggle:** el-switch, updates `active` boolean
- **Sort:** up/down arrow buttons to reorder (updates `sort_order`). Each button must have an `aria-label` attribute (e.g. `aria-label="上移"` for the up arrow, `aria-label="下移"` for the down arrow) for accessibility.

### 8. TodoInvitePage (new page — reuse RegisterPage pattern)
- **Route:** `/todo/:id/invite?token=xxx` (D-09-10)
- **Pattern:** Follow `RegisterPage.vue` exactly — token validation, loading, error, success states
- **Layout:** max-width 480px, centered, 12px card radius
- **Form:** Readonly display of todo title + description; editable fields for collaborator to fill
- **Fields:** Defined per todo type (not specified here — per-plan detail)
- **No sensitive data:** Collaborator sees only fields they're invited to fill (D-09-09)
- **Success copy:** "信息已提交，感谢您的配合"

### 9. StatsCard (reused from existing patterns)
- **Purpose:** Show completed count / total count below each ring chart
- **Typography:** 22px/700 number, 13px/400 label (deadline/tertiary info merged into 13px label tier)
