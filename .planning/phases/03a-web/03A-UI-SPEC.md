## Spacing Scale

继承 LoginView.vue 现有间距规范（8px 基准倍数）：

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | code-row gap, icon-inline |
| sm | 8px | Flex gap in code-row (gap: 8px) |
| md | 16px | login-card 内间距, 移动端 card padding |
| lg | 24px | login-card 桌面端 padding, brand 下方间距 |
| xl | 32px | — (未使用) |
| 2xl | 48px | — (未使用) |
| 3xl | 64px | — (未使用) |

**Exceptions**: 按钮高度固定 44px（触屏可及性标准），非 4px 倍数。

## Typography

| Role | Size | Weight | Line Height | Usage |
|------|------|--------|-------------|-------|
| Brand heading | 24px | 700 (bold) | 1.2 | 品牌标题「易人事」 |
| Body | 14px | 400 | 1.5 | 副标题、占位符、提示文案 |
| Button | 16px | 400 | 1.5 | 登录/注册主按钮文字 |
| Caption | 12px | 400 | 1.5 | copyright 版权信息 |

**Note**: 权重最多使用 2 种（400 regular + 700 bold），移动端 brand h1 缩小至 20px，body 缩小至 12px。

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | #1677ff | 页面背景（login-page）、品牌色 |
| Secondary (30%) | #ffffff | 登录卡片背景（login-card）、表单输入框背景 |

| Text light | #bbbbbb | copyright 文案 |

**Accent reserved for**:
- el-tabs 激活标签下划线（Element Plus CSS 变量覆盖）
- type="primary" el-button 主按钮
- 表单输入框 focus 状态边框

## Copywriting Contract

| Element | Copy | Source |
|---------|------|--------|
| 注册 Tab 标签 | 注册 | D-01: 替换 Tab 3「微信登录」为「注册」Tab |
| 注册按钮 | 注册 | D-03: 区别于 Tab 1 的「登录」 |
| 获取验证码按钮（默认） | 获取验证码 | 继承 Tab 1 现有文案 |
| 获取验证码按钮（倒计时中） | 已发送({countdown}s) | 继承 Tab 1 现有文案 |
| 发送成功提示 | 验证码已发送 | ElMessage.success 继承 |
| 验证码发送失败提示 | 发送失败 / 后端错误消息 | ElMessage.error 继承 |
| 验证码输入为空提示 | 请填写手机号和验证码 | 继承 handleSmsLogin 逻辑（复用 Tab 1 表单） |
| 注册提交失败提示 | 后端错误消息 | 继承 handleSmsLogin 错误处理模式 |
| 403 禁止登录提示 | 您的账号为员工账号，请使用员工端微信小程序登录 | 继承 Tab 1 现有逻辑 |
| 跳转路由 | /onboarding/org-setup | D-04: onboarding_required=true 时跳转 |

**Empty state**: 不适用 — 注册 Tab 始终有表单可见，无数据为空场景。

**Destructive actions**: 无破坏性操作（本 phase 仅新增注册 Tab）。

**Deferred UX gap**（不在本 phase 处理）:
- 已注册手机号输入后点击注册 → 后端视为登录 → 直接跳转 /home，无"该手机号已注册"提示
- 归因：D-11 明确延期
