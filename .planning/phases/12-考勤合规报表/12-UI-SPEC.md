## Visual Hierarchy

Declared focus order (primary to tertiary):

| Level | Element | Role |
|-------|---------|------|
| 1 (primary) | `.page-title` | Single focal point — page name, employee name, or report title |
| 2 (secondary) | Stat overview cards | At-a-glance numbers that answer "how are we doing?" |
| 3 (tertiary) | Table data | Detailed records, supporting the stat cards |

## Typography

**4 sizes, 2 weights** (reduced from 7 per checker D-4 feedback):

| Role | Size | Weight | Line Height | Notes |
|------|------|--------|-------------|-------|
| Page title / stat value | 28px | 700 | 1.2 | `.page-title` from variables.scss; stat numbers share size for visual balance |
| Body / table / button | 14px | 400 | 1.5 | Default Element Plus body; button text inherits this size |
| Table header / labels | 13px | 600 | — | `font-weight: 600` in `header-cell-style` |
| Caption / tertiary labels | 12px | 400 | — | Stat labels, input labels, table column sub-labels |

**Font stack:** `'Inter', -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, 'Helvetica Neue', Arial, sans-serif`

## Color

**Ratio: 60% dominant / 30% secondary / 10% accent** (per checker D-3 feedback):

| Role | Value | Ratio | Usage |
|------|-------|-------|-------|
| Dominant (60%) | `#FAFBFC` | 60% | Page background (`--bg-page`) |
| Secondary (30%) | `#FFFFFF` (95% opacity glass cards) | 30% | Table, stat cards, dialogs |
| Accent (10%) | `#7C3AED` | 10% | Primary actions only — reserved list below |
| Success | `#10B981` | — | Normal clock-in, good attendance rate (>= 95%) |
| Warning | `#F59E0B` | — | Late clock, attendance rate 80-95% |
| Danger | `#EF4444` | — | Anomaly rows (late > 3 or absent > 1 day), missing clock, attendance rate < 80% |

**Accent reserved for (10%):**
- Primary CTA button: "导出 Excel"
- Active sidebar menu item
- el-tag type="danger" for anomaly rows

**Anomaly highlight rule (from D-12-10):**
- `el-tag type="danger"` on employee row if: 迟到次数 > 3 OR 缺勤天数 > 1
- Background tint on table row: `rgba(239, 68, 68, 0.04)`

**Color tokens from `variables.scss`:**
- `--primary: #7C3AED` / `--primary-hover: #8B5CF6` / `--primary-light: #A78BFA`
- `--success: #10B981`
- `--warning: #F59E0B`
- `--danger: #EF4444`
- `--text-primary: #1F2937` / `--text-secondary: #6B7280` / `--text-tertiary: #9CA3AF`

## Component Inventory

### Layout Shell
- Reuse `AppLayout.vue` sidebar structure with new sub-menu group
- `el-sub-menu` with icon + title "合规报表"
- 4 `el-menu-item` children under `/compliance/` sub-paths

### Stat Overview Cards
Reuse pattern from `AttendanceMonthly.vue` (`.stats-grid`, `.stat-card`, `.stat-icon`):
- 4-column grid on desktop, 2-column on tablet, stacked on mobile
- Gradient background on stat-icon per category (success/warning/danger/primary)
- Icon: 48x48px, `border-radius: 12px`, `font-size: 22px`

### Data Table
Reuse pattern from `AttendanceMonthly.vue` and `ClockLive.vue`:
- el-table with `stripe`, `modern-table` class
- Header cell style: `background: #F9FAFB, color: #374151, font-weight: 600`
- Row hover: `rgba(var(--primary), 0.02)`
- Fixed left column for employee name
- Fixed right column for actions (view detail, export)
- Pagination: `total, prev, pager, next` at bottom right

### Filter Bar
- el-date-picker (type="month") for month selection — 150px width, `size="large"`
- el-select (department filter) — 160px width
- el-input with search icon for keyword search — 240px width
- All filters on single row in page-header `.header-actions`

### Export Button
- Primary CTA: `el-button type="primary" size="large"`
- Icon: `<Download />` from `@element-plus/icons-vue`
- Label: "导出 Excel"

### Empty State
- `.empty-state.glass-card` centered content
- 72x72px icon circle with gradient background
- Heading: 14px bold (merged into body size with weight 700)
- Body: 14px tertiary
- Optional action button

### Anomaly Row Highlight
- `el-tag type="danger"` showing anomaly count in dedicated column
- Table row background: `rgba(239, 68, 68, 0.04)` when anomaly tag present

## Page Layouts

All 4 compliance report pages follow identical layout structure:

```
.page-view
  .page-header
    .header-left
      h1.page-title     ← Page-specific title (28px, weight 700, level 1 in visual hierarchy)
      p.page-subtitle   ← Selected month/dept summary (14px, weight 400)
    .header-actions
      el-date-picker    ← Month selector
      el-select         ← Department filter
      el-button primary ← "导出 Excel"
  .stats-overview       ← 4 stat cards (28px stat values, 12px labels — level 2)
  .table-container.glass-card
    el-table (data)     ← 14px body, 13px headers — level 3
    .pagination-wrapper
  .empty-state (if no data)
```

**Page titles:**
1. 加班统计报表 — "加班统计"
2. 请假合规报表 — "请假合规"
3. 出勤异常报表 — "出勤异常"
4. 月度考勤汇总 — "月度汇总"

## API Contract (frontend)

Reports API endpoint structure (backend generates Excel via excelize, frontend triggers download):

```
GET /api/v1/compliance/overtime?year_month=YYYY-MM&dept_ids=1,2,3
GET /api/v1/compliance/leave?year_month=YYYY-MM&dept_ids=1,2,3
GET /api/v1/compliance/anomaly?year_month=YYYY-MM&dept_ids=1,2,3
GET /api/v1/compliance/monthly/export?year_month=YYYY-MM&dept_ids=1,2,3
GET /api/v1/compliance/monthly?year_month=YYYY-MM&dept_ids=1,2,3&page=1&page_size=20
```

All endpoints return `Blob` for export, `ApiResponse<PageData<T>>` for table load.

## Interaction Patterns

### Filter + View Flow (all 4 pages)
1. **Select month** (el-date-picker, default: current month)
2. **Select department** (el-select, default: all)
3. **Click "查看报表"** or auto-load on filter change
4. Table populates with compliance data

### Export Flow (all 4 pages)
1. Click "导出 Excel" button
2. Frontend calls export API with current filters
3. Backend returns Blob via `application/vnd.openxmlformats-officedocument.spreadsheetml.sheet`
4. Frontend creates `<a>` download link, triggers click
5. `ElMessage.success('导出成功')` on completion

### Anomaly Row Interaction (COMP-07 page only)
- Rows with anomaly count > 0 show `el-tag type="danger"` in dedicated column
- Optional: click row to open detail drawer with anomaly breakdown

## Accessibility Notes

- All filter controls labeled (aria-label or associated label)
- Table headers use `scope="col"`
- Export button: `aria-label="导出 Excel"`
- Color not sole indicator — always pair color with text/icon (e.g., `el-tag type="danger"` with "迟到 4 次" text)
