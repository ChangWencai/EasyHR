    2. 创建 internal/common/crypto/aes.go (per D-15):
       - `func Encrypt(plaintext string, key []byte) (string, error)`:
         - aes.NewCipher(key) -> cipher.NewGCM(block)
         - nonce: make([]byte, gcm.NonceSize()), io.ReadFull(rand.Reader, nonce)
         - gcm.Seal(nonce, nonce, []byte(plaintext), nil)
         - base64.StdEncoding.EncodeToString(ciphertext)
         - key 必须 32 字节 (AES-256)
       - `func Decrypt(encoded string, key []byte) (string, error)`:
         - base64.StdEncoding.DecodeString(encoded)
         - aes.NewCipher(key) -> cipher.NewGCM(block)
         - nonceSize := gcm.NonceSize()
         - gcm.Open(nil, data[:nonceSize], data[nonceSize:], nil)
         - return string(plaintext), nil
       - 使用标准库 crypto/aes + crypto/cipher，不引入第三方库

    3. 创建 internal/common/crypto/hash.go:
       - `func HashSHA256(input string) string`:
         - sha256.Sum256([]byte(input))
         - hex.EncodeToString(sum[:])
       - 使用标准库 crypto/sha256 + encoding/hex
