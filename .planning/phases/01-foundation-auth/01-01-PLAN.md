    1. 创建 `internal/common/crypto/aes.go` per D-15：
       - `Encrypt(plaintext string, key []byte) (string, error)`：AES-256-GCM 加密，随机 nonce，返回 base64 编码字符串。key 必须为 32 字节。
       - `Decrypt(encoded string, key []byte) (string, error)`：base64 解码 → 提取 nonce → GCM 解密 → 返回原文。
       - 错误处理：key 长度不正确、base64 解码失败、解密失败均返回明确错误。

    2. 创建 `internal/common/crypto/hash.go`：
       - `HashSHA256(input string) string`：返回 hex 编码的 SHA-256 哈希值。
       - 用于手机号、身份证号等敏感字段的索引列，支持精确查询而不暴露原文。

    3. 创建 `internal/common/crypto/mask.go` per D-17：
       - `MaskPhone(phone string) string`：手机号保留前3后4，中间用 **** 替换。如 "138****8000"。输入非法长度返回原值。
       - `MaskIDCard(idCard string) string`：身份证号保留前6后4，中间用 **** 替换。如 "110101****1234"。输入非法长度返回原值。

    4. 创建 `internal/common/crypto/crypto_test.go`：测试上述所有函数的行为场景。

    5. 创建 `internal/common/middleware/tenant.go` per D-09/D-10：
       - `TenantScope(orgID int64) func(db *gorm.DB) *gorm.DB`：返回 GORM Scope 函数，自动追加 `WHERE org_id = ?`。
       - `TenantMiddleware() gin.HandlerFunc`：从 `gin.Context` 获取 `org_id`（由 Auth 中间件设置），注入到 context 中供 Repository 层读取。注意：此中间件不处理认证（认证中间件在 Plan 02），仅负责 org_id 上下文传递。

    6. 创建 `internal/common/middleware/cors.go`：Gin CORS 中间件，允许 `*` Origin（开发阶段），Methods: GET/POST/PUT/DELETE/PATCH，Headers: Content-Type/Authorization，MaxAge: 12h。

    7. 创建 `internal/common/middleware/ratelimit.go`：基于 Redis 的滑动窗口限流中间件。`RateLimit(rdb *redis.Client, limit int, window time.Duration, keyFunc func(*gin.Context) string) gin.HandlerFunc`。默认按 IP 限流。

    8. 创建 `internal/common/middleware/logger.go`：请求日志中间件，记录 method、path、status、latency、client_ip。使用 Zap。

    9. 创建 `internal/common/middleware/middleware_test.go`：测试 TenantScope、CORS 头、RateLimit 限流效果。

    10. 创建 `internal/common/model/org.go`：Organization 模型，嵌入 BaseModel，字段：Name（企业名称）、CreditCode（统一社会信用代码）、City（城市）、ContactName（联系人）、ContactPhone（联系人电话，加密列+哈希列）、Status（active/inactive）。

    11. 创建 `internal/common/model/user.go`：User 模型，嵌入 BaseModel，字段：Phone（加密列）、PhoneHash（哈希列，唯一索引）、Name、Role（owner/admin/member）、Status（active/inactive）、OrgID（关联企业）。使用 GORM struct tag 标记加密列类型为 `varchar(200)`，哈希列为 `varchar(64)`。

    12. 更新 `cmd/server/main.go`：注册 CORS、请求日志中间件。注意不注册 Auth 中间件（Plan 02 实现）。
