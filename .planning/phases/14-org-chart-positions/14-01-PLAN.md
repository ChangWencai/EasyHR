## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-14-01 | Tampering | Position.name | mitigate | max 100 chars validated via `binding:"required,max=100"` in DTO; org isolation via TenantScope in repository |
| T-14-02 | Tampering | Position uniqueness | mitigate | `ExistsByNameAndDept` checks before create/update; handles NULL department_id with explicit `IS NULL` query |
