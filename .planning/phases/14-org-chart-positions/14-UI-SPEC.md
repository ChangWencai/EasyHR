## Component Inventory

### OrgChart (ECharts Tree)

**Department nodes:**
- Shape: rounded rect, color #7C3AED, text #FFFFFF, font-size 13px weight 400
- Icon: el-icon-office-building (or Folder element-plus icon)
- Hover: slight brightness increase (filter: brightness(1.1))
- Draggable: cursor changes to grab, on drag start node border becomes #5B21B6 2px
- Inline edit: on click, node label becomes `<input>` with border #7C3AED, confirm on blur/Enter

**Position nodes:**
- Shape: rounded rect, color #A78BFA, text #FFFFFF, font-size 12px weight 400
- Icon: el-icon-briefcase
- Not draggable (positions belong to departments)

**Employee nodes:**
- Shape: rounded rect, color #F59E0B, text #FFFFFF, font-size 12px weight 400
- Icon: el-icon-user
- Not draggable

**Tree edges:**
- Style: color #D9D9D9, width 1px

**Search highlight (D-14-08):**
- Matching nodes: itemStyle color stays at assigned color, label color becomes #4F6EF7, fontWeight 600
- Non-matching nodes: opacity 0.25 (chart option opacity)

### Delete Transfer Dialog (el-dialog)

**Layout:** single column, width 480px
- Header: title "删除部门「{name}」", destructive red title color
- Body:
  1. Alert box (el-alert type="warning"): "{N} 名员工将被转移至其他部门"
  2. Employee list (el-table, max-height 240px, scrollable): columns: 姓名/手机号/岗位
  3. Form item: 接收部门 (el-select with dept options, required)
- Footer: 取消 (text button) + 确认转移并删除 (el-button type="danger", requires dept selected)

### Position Dropdown (el-select in employee form)

**Grouping (D-14-13, D-14-14, D-14-15):**
- Group 1 label: "部门专属岗位" (shown only when current dept has dept-specific positions)
- Group 1 options: dept-specific positions (filtered by current department_id)
- Group 2 label: "通用岗位" (shown always)
- Group 2 options: common positions (department_id = NULL)
- Unassigned option: "未分配岗位" (value = null, shown at top of list before groups)
- When department_id is null: show only group 2 (通用岗位) + unassigned option
- el-select prop: `group-style` via slot structure or `el-optgroup`

**Visual distinction:**
- Group label color: var(--text-secondary) #6B7280, font-size 12px
- Selected item: uses element-plus default highlight

### Position Management Dialog

**Layout:** el-dialog width 520px, create/edit position form
- Fields: 岗位名称 (el-input, required), 所属部门 (el-select with dept options, optional -- null = common), 排序 (el-input-number)
- Footer: 取消 + 确认

### Inline Edit (Dept Node)

- Trigger: click on department node label in ECharts
- UX: text input replaces label, styled with el-input mini (padding: 4px 8px, font-size: 13px)
- Confirm: blur or Enter key
- Cancel: Escape key
- Save API: PATCH /api/departments/:id with { name: "new_name" }
- Error: ElMessage.error("保存失败，请重试"), revert label
