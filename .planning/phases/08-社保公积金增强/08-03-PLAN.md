## UI-SPEC Design Contract (key sections)

### StopDialog Fields (UI-SPEC §4):
- 员工姓名: el-select filterable (auto-filled if opened from row)
- 身份证号: el-input readonly (auto-filled)
- 终止月份: el-date-picker type="month" (min = current month, D-SI-08)
- 减员原因: el-select 三选一: 跳槽 / 退休 / 其他
- 转出日期: el-date-picker (triggers SI-13 tooltip)

- 每月5日前转出 -> 当月生效
- 5-25日转出 -> 次月生效
- 25日后转出 -> 下下月生效
