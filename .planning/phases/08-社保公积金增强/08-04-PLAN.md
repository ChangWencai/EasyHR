## D-SI-13 Excel Export Requirements

导出列（从左到右）：
1. 员工姓名
2. 城市
3. 社保基数
4. 参保月
5. 缴费渠道（自主缴费/代理新客/代理已合作）
6. 状态（正常/待缴/欠缴/已转出/未转出）
7. 五险分项（6险 × 2列 = 12列）：
   - 养老保险_单位
   - 养老保险_个人
   - 医疗保险_单位
   - 医疗保险_个人
   - 失业保险_单位
   - 失业保险_个人
   - 工伤保险_单位
   - 工伤保险_个人
   - 生育保险_单位
   - 生育保险_个人
   - 住房公积金_单位
   - 住房公积金_个人
8. 合计单位（= SUM 6险单位列）
9. 合计个人（= SUM 6险个人列）
10. 其他缴费：滞纳金/残保金/漏缴/补缴（若有）

   ```go
   func translateChannel(channel string) string {
       switch channel {
       case SIPayChannelSelf: return "自主缴费"
       case SIPayChannelAgentNew: return "代理缴费新客"
       case SIPayChannelAgentExisting: return "代理缴费已合作"
       default: return channel
       }
   }

   func translateStatus(status string) string {
       switch status {
       case SIPayStatusNormal: return "正常"
       case SIPayStatusPending: return "待缴"
       case SIPayStatusOverdue: return "欠缴"
       case SIPayStatusTransferred: return "已转出"
       case SIPayStatusNotTransferred: return "未转出"
       default: return status
       }
   }
   ```
