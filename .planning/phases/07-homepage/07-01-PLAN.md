    **model.go** — Data structures:
    ```go
    package dashboard

    type TodoType string

    const (
        TodoSocialInsurance  TodoType = "social_insurance"
        TodoTax             TodoType = "tax"
        TodoEmployee        TodoType = "employee"
        TodoContract        TodoType = "contract"
        TodoExpense         TodoType = "expense"
        TodoVoucher         TodoType = "voucher"
    )

    type TodoItem struct {
        Type     TodoType `json:"type"`
        Title    string   `json:"title"`
        Count    int      `json:"count"`
        Deadline string   `json:"deadline,omitempty"`
        Priority int      `json:"priority"`
    }

    type Overview struct {
        EmployeeCount         int    `json:"employee_count"`
        JoinedThisMonth       int    `json:"joined_this_month"`
        LeftThisMonth         int    `json:"left_this_month"`
        SocialInsuranceTotal  string `json:"social_insurance_total"`
        PayrollTotal          string `json:"payroll_total"`
    }

    type DashboardResult struct {
        Todos    []TodoItem `json:"todos"`
        Overview Overview   `json:"overview"`
    }
    ```

    **repository.go** — DashboardRepository interface + implementation:
    - Define `DashboardRepository` interface with 7 methods: GetEmployeeStats, GetPayrollTotal, GetSocialInsuranceTotal, GetPendingVouchers, GetPendingExpenses, GetTaxReminders, GetContractExpirations. Each accepts (ctx context.Context, orgID uint).
    - `NewRepository(db *gorm.DB) *DashboardRepositoryImpl` — store db reference.
    - `GetEmployeeStats`: query employee table where org_id=? and status IN ('active','probation') for active count; created_at in current month for joined; status='resigned' and leaving_date in current month for left. Use soft delete filter: deleted_at IS NULL.
    - `GetPayrollTotal`: query payroll_records table where org_id=? and status='paid' and period_id corresponds to current month; sum net_salary. Return "0" if no records.
    - `GetSocialInsuranceTotal`: query social_insurance_records where org_id=? and payment_month=current month; sum total_amount (personal + employer). Return "0" if table doesn't exist yet (Phase 3 not executed) — use `db.Exec` with IF NOT EXISTS pattern or just query and catch error.
    - `GetPendingVouchers`: query vouchers where org_id=? and status='submitted' (Phase 6). Return 0 if table doesn't exist.
    - `GetPendingExpenses`: query expense_reimbursements where org_id=? and status='pending'. Return 0 if table doesn't exist.
    - `GetTaxReminders`: query tax_reminders where org_id=? and status='pending' and deadline within 3 days. Return 0 if table doesn't exist.
    - `GetContractExpirations`: query contracts where org_id=? and expiry_date within 30 days and status='active'. Return 0 if table doesn't exist.
    - All queries use TenantScope via orgID param (not middleware scope). Queries should not crash if the table doesn't exist — use `db.Migrator().HasTable()` check or try/catch style.

    **service.go** — DashboardService:
    ```go
    type DashboardService struct {
        repo DashboardRepository
    }
    func NewService(repo DashboardRepository) *DashboardService
    func (s *DashboardService) GetDashboard(ctx context.Context, orgID uint) (*DashboardResult, error)
    ```
    - GetDashboard collects all data concurrently using goroutines + errgroup.
    - Builds []TodoItem only for types with count > 0.
    - Sorts todos by Priority ascending.
    - Returns zero Overview if any query fails (partial data is acceptable per D-13: "data only on entry").

    **handler.go** — HTTP handler:
    ```go
    func RegisterDashboardRouter(rg *gin.RouterGroup, svc *DashboardService, authMiddleware gin.HandlerFunc)
    ```
    - GET /dashboard handler: extract orgID from c.Get("org_id") (set by JWT middleware). Call svc.GetDashboard. Return 200 with JSON. Handle errors with response.Error.

    **router.go** — Router registration:
    ```go
    func RegisterDashboardRouter(rg *gin.RouterGroup, authMiddleware gin.HandlerFunc, db *gorm.DB) {
        repo := NewRepository(db)
        svc := NewService(repo)
        handler := NewHandler(svc)
        rg := rg.Group("/dashboard")
        rg.Use(authMiddleware)
        rg.GET("", handler.GetDashboard)
    }
    ```
