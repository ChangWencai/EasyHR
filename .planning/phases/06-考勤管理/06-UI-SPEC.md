## Interaction Patterns

### 打卡规则 Tab 切换

- 顶部 el-tabs 点击切换，每次切换保留各 Tab 表单状态（不重置）
- 切换前自动保存当前 Tab 内容（防丢数据）
- 节假日列表支持增删，添加行默认空值，删除需 el-popconfirm 二次确认

### 打卡实况点击行展开

- 点击行展开 el-popover（trigger=click，placement=top）
- 点击其他行或点击关闭按钮关闭当前 Popover
- Popover 展示请假/出差/外出/加班等统计，按 approved/pending 分类
- 仅展示当月数据

### 打卡状态颜色标注

- 表格打卡时间列文字颜色实现：
  - 正常（normal）: color=#52C41A
  - 迟到（late）: color=#FA8C16
  - 缺勤（absent）: color=#FF4D4F
  - 未排班（no_schedule）: color=#BFBFBF（不显示打卡时间，显示"--"）
- 使用 el-tag 的 type 属性映射 + 自定义 CSS 变量控制颜色
- 或使用 `:class` 绑定文字颜色，不使用 el-tag 背景色

### 审批列表操作

- 同意：el-popconfirm 确认后调用 approve API，成功后刷新列表 + ElMessage.success
- 驳回：弹出 el-dialog（复用 OffboardingList 驳回 dialog 模式），输入驳回原因（选填），确认后调用 reject API
- 撤销：el-popconfirm 确认后调用 cancel API（仅我发起的 Tab 显示）

### 新建申请时长自动计算

- 视图状态不在 URL 参数中持久化（刷新后恢复默认统计视图）
- 导出月报调用后端 Excel 导出 API，文件下载

### 节假日列底色（格子视图）

- 节假日日期由打卡规则中的节假日列表提供
- 格子矩阵表列头组件接收节假日 date[]，判断当日是否为节假日
- 是：列头文字红色，列背景 #FFF1F0，列所有单元格背景 #FFF1F0
- 否：默认白色背景
