### Step 1f — Replace handleSign with handleConfirm

Find and replace the `handleSign` function:
```typescript
async function handleConfirm() {
  signing.value = true
  try {
    await confirmSlip(token)
    confirmed.value = true
    if (slip.value) {
      slip.value.confirmed_at = new Date().toLocaleString('zh-CN')
    }
    ElMessage.success('工资条已确认')
  } catch {
    ElMessage.error('确认失败，请重试')
  } finally {
    signing.value = false
  }
}
```
