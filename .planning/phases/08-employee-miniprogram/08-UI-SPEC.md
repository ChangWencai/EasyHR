# Phase 08 — UI Design Contract

> Visual and interaction contract for Phase 08 员工微信小程序 (WeChat Mini Program). Generated by gsd-ui-researcher, verified by gsd-ui-checker.

---

## Design System

| Property | Value | Source |
|----------|-------|--------|
| Tool | none | WeChat Mini Program, not web |
| Component library | WeUI | D-25 (08-CONTEXT.md) |
| Icon library | WeUI built-in icons | D-25 |

| `input` / `textarea` | 文本输入 |

| `uploader` | 票据照片上传（最多9张） |

## Spacing Scale

继承 ui-ux.md §1.3 视觉规范：

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | icon gaps, inline padding |
| sm | 8px | 紧凑元素间距 |
| md | 16px | 默认元素间距、页面边距 |
| lg | 24px | 区块间距、卡片内间距 |
| xl | 32px | 页面级间距 |

Exceptions:
- 底部 Tab 栏高度固定 `56px`，需叠加 `safe-area-inset-bottom`
- 顶部状态栏高度 `statusBarHeight`（微信 `wx.getSystemInfo` 获取）
- 表单输入框最小触控区域 `44px`（iOS 无障碍标准）
- 票据照片上传按钮：`80px × 80px`，最大 9 张

## Typography

继承 ui-ux.md §1.3（微信 webview 系统字体）：

| Role | Size | Weight | Line Height | Usage |
|------|------|--------|-------------|-------|
| Heading | 18px | 600 | 1.3 | 模块标题、Tab 名称、页面大标题、金额数字 |
| Body | 16px | 400 | 1.5 | 正文、列表项、说明文字 |
| Caption | 14px | 400 | 1.5 | 辅助文字、空状态、日期 |
| Mini | 12px | 400 | 1.4 | 标签、徽章、权限说明 |

> 微信小程序不支持自定义字体，使用 webview 系统默认字体。无衬线（iOS San Francisco / Android Roboto）。

## Interaction Contracts

### 敏感信息验证流程（工资条明细）

来源：08-CONTEXT.md D-06, D-27b

```
1. 用户点击月份卡片
2. 页面展示摘要（应发合计、实发合计）→ 按钮"查看明细"
3. 点击"查看明细" → 弹出短信验证弹窗
4. 用户输入手机号后4位 + 短信验证码
5. 后端验证通过 → 返回完整明细 → 展示明细页
6. 验证通过后当月无需再验证（前端缓存标记）
```

### Tab 切换

- 使用微信 `wx.switchTab` API
- 无需页面过渡动画（touch only 设备无感知延迟问题）
- 切换时重新加载该 Tab 的数据（不清除其他 Tab 的数据）

### 票据照片上传

来源：08-CONTEXT.md D-18

```
1. 后端生成 OSS 预签名上传 URL
2. 前端直接上传图片到 OSS（不走服务端中转）
3. 上传成功后记录 OSS file key 到表单
4. 提交表单时携带所有 file key 列表
5. 支持拍照和相册选择（wx.chooseMedia）
```

### 报销提交确认

来源：ui-ux.md §4.3 状态反馈规范

- 提交前显示确认弹窗（金额 + 类型）
- 提交成功后跳转至报销列表页
- 提交失败显示红色 toast，不跳转

### 空状态展示

所有列表页（工资条/合同/社保/报销）：

```
┌─────────────────────────────────┐
│                                 │
│         [插画占位 120x120]       │
│                                 │
│      暂无{功能}记录              │
│                                 │
│   {对应的引导文案}              │
│                                 │
└─────────────────────────────────┘
```

## Component States

### 列表项状态

| 状态 | 视觉处理 |
|------|----------|
| 默认 | 白色背景，左侧蓝色指示条 |
| 加载中 | 骨架屏（skeleton）或 spinner |
| 无数据 | 空状态插画 + 文案 |
| 网络错误 | 重试按钮 + 错误文案 |

### 表单状态

| 状态 | 视觉处理 |
|------|----------|
| 默认 | 占位符文字，灰色边框 |
| 聚焦 | 蓝色边框，label 上浮 |
| 填写中 | 黑色文字 |
| 填写完成 | 正常显示 |
| 验证失败 | 红色边框，红色错误文案在输入框下方 |
| 禁用 | 灰色背景，不可点击 |

### 按钮状态

| 状态 | 视觉处理 |
|------|----------|
| 默认 | 主色背景，白色文字 |
| 加载中 | 主色背景，显示 spinner，文字变为"处理中..." |
| 禁用（未填写完） | 灰色背景，不可点击 |
| 危险操作（删除/取消） | 红色边框，红色文字，无背景色 |
