# Requirements: v1.3 产品功能全面优化（基于 PRD 1.1）

**Milestone:** v1.3
**Defined:** 2026-04-17
**Goal:** 根据 PRD 1.1 对现有产品进行功能优化和补全，新增待办中心、考勤管理、完善薪资/社保/员工管理模块

## 2. 考勤管理（ATT）

### 打卡设置

- [ ] **ATT-01**: 管理员可设置固定时间上下班模式（选择上班日+上下班时间+节假日+打卡位置+打卡方式）
- [ ] **ATT-02**: 管理员可设置按排班上下班模式（自定义班次名称/时间+排班周期+员工分配+特殊日期）
- [ ] **ATT-03**: 管理员可设置自由上下班模式（选择工作日+打卡时间+节假日+打卡位置+打卡方式）
- [ ] **ATT-04**: 打卡设置默认法定节假日不用打卡

### 今日打卡实况

- [ ] **ATT-05**: 管理员可查看今日员工打卡实况列表（员工-打卡时间点）
- [ ] **ATT-06**: 管理员可邀请员工点签打卡（转发链接）
- [ ] **ATT-07**: 点击员工可查看假勤统计弹窗（实际出勤+加班+请假+缺勤，可选定年月查看）
- [ ] **ATT-08**: 管理员可手动修改员工假勤统计数据

### 审批流

- [ ] **ATT-09**: 员工可提交补卡申请（开始/结束时间+时长自动计算+事由+附件+审批人+抄送人）
- [ ] **ATT-10**: 员工可提交调班申请（开始/结束时间+时长+事由+审批人+抄送人）
- [ ] **ATT-11**: 员工可提交出差申请（开始/结束时间+时长+事由+审批人+抄送人），不影响打卡
- [ ] **ATT-12**: 员工可提交外出申请（规则同出差）
- [ ] **ATT-13**: 员工可提交请假申请，支持7种类型：事假（无薪）、病假（按工龄系数）、调休假（带薪）、年假（带薪）、婚假（带薪）、产假（带薪）、陪产假（带薪）
- [ ] **ATT-14**: 请假申请自动计算请假时长（小时）
- [ ] **ATT-15**: 管理员可同意或驳回审批申请
- [ ] **ATT-16**: 审批列表展示待办条数，仅展示与考勤相关的申请
- [ ] **ATT-17**: 请假附件支持拍照上传（必填或非必填可配置）

### 出勤月报

- [ ] **ATT-18**: 管理员可查看出勤月报（默认最近一个月，可选定年月）
- [ ] **ATT-19**: 出勤月报显示每个员工的实际出勤、应出勤、加班时长
- [ ] **ATT-20**: 管理员可查看员工每日打卡记录详情
- [ ] **ATT-21**: 出勤月报支持 Excel 格式导出（含每日打卡记录）

## 3. 薪资管理增强（SAL）

### 数据看板

- [ ] **SAL-01**: 薪资数据看板展示当月应发总额（基本+绩效+补贴+奖金+加班费+年终奖+扣款），环比上月百分比
- [ ] **SAL-02**: 薪资数据看板展示当月实发工资总额，环比上月百分比
- [ ] **SAL-03**: 薪资数据看板展示当月社保公积金总额（单位+个人合计），环比上月百分比
- [ ] **SAL-04**: 薪资数据看板展示当月个税总额，环比上月百分比

### 调薪/普调

- [ ] **SAL-05**: 管理员可对选定员工进行调薪（岗位工资/补贴/奖金/年终奖/其他扣除/生效期限），支持输入金额或比例
- [ ] **SAL-06**: 管理员可对选定部门进行普调（调整规则同调薪）
- [ ] **SAL-07**: 调薪数据按生效期限自动应用于当月工资核算

### 个税上传

- [ ] **SAL-08**: 管理员可上传个税 Excel 文件，系统自动抓取关键字段（员工姓名-个税-应退/应补额）
- [ ] **SAL-09**: 个税上传成功后自动更新当月工资表中个税数据
- [ ] **SAL-10**: 个税上传失败时提示上传失败原因或重新上传

### 绩效系数

- [ ] **SAL-11**: 管理员可为选定员工设置绩效系数（0%-100%）
- [ ] **SAL-12**: 绩效系数自动挂钩绩效工资计算（绩效工资 = 绩效工资标准 × 绩效系数）

### 薪资算法增强

- [ ] **SAL-13**: 基本工资按计薪天数计算（基本工资/应出勤×计薪天数+病假工资），计薪天数=实际出勤+法定节假日+带薪假天数
- [ ] **SAL-14**: 病假工资按工龄系数计算（6个月医疗期内/超6个月）
- [ ] **SAL-15**: 加班费按法定系数计算（工作日150%/双休日200%/节假日300%）
- [ ] **SAL-16**: 管理员可查看税前工资明细（基本工资/绩效工资/补贴/奖金/加班费/年终奖/其他扣除分项展示）

### 发工资条

- [ ] **SAL-17**: 管理员可向全员发送当月工资条通知
- [ ] **SAL-18**: 管理员可向选定员工发送指定月份工资条

### 下载表单

- [ ] **SAL-19**: 薪资列表支持 Excel 格式下载（当前页/含税前明细）

## 4. 社保公积金增强（SI）

### 数据看板

- [ ] **SI-01**: 社保数据看板展示当月应缴总额（单位+个人+欠缴），环比上月百分比
- [ ] **SI-02**: 社保数据看板展示当月单位部分合计，环比上月百分比
- [ ] **SI-03**: 社保数据看板展示当月个人部分合计，环比上月百分比
- [ ] **SI-04**: 社保数据看板展示欠缴金额（上月+当月截止25日未缴），26日更新数据

### 增员优化

- [ ] **SI-05**: 增员弹窗支持姓名输入或检索，根据姓名自动检索身份证号码
- [ ] **SI-06**: 增员弹窗设置起始月份（默认与预入职月份同步，可选近3个月），当月1日前生成欠缴账单
- [ ] **SI-07**: 增员弹窗支持输入社保缴费城市和缴费基数
- [ ] **SI-08**: 增员弹窗支持输入公积金缴费比例和基数（默认城市与社保同步）

### 减员优化

- [ ] **SI-09**: 减员弹窗支持姓名输入或检索，根据姓名自动检索身份证号码
- [ ] **SI-10**: 减员弹窗设置终止月份（默认与离职月份同步，不可早于当月）
- [ ] **SI-11**: 减员弹窗必填原因（三选一）
- [ ] **SI-12**: 减员弹窗支持选择转出社保日期和封存公积金日期，自动生成最后缴存月
- [ ] **SI-13**: 转出日期提示：每月5-26日前转出当月生效，26日后次月生效

### 缴费渠道与状态管理

- [ ] **SI-14**: 管理员可选择缴费渠道（自主缴费/代理缴费新客/代理缴费已合作）
- [ ] **SI-15**: 自主缴费点击跳转已缴完成确认
- [ ] **SI-16**: 代理缴费（已合作）每月15日自动从平台充值额扣缴，成功/失败发送通知
- [ ] **SI-17**: 缴费状态自动流转（正常→待缴→欠缴→已转出），26日为状态更新分界点
- [ ] **SI-18**: 社保列表新增5种状态（正常/待缴/欠缴/已转出/未转出）

### 提醒与明细

- [ ] **SI-19**: 社保关键节点红字提醒，展示社保政策通知
- [ ] **SI-20**: 社保明细展示社保缴费城市/缴费基数/五险各项金额（单位+个人）/公积金（单位+个人）/其他缴费（滞纳金/残保金/漏缴/补缴）
- [ ] **SI-21**: 社保列表支持 Excel 格式下载导出

## 5. 员工管理增强（EMP）

### 数据看板

- [ ] **EMP-01**: 员工数据看板展示在职人数、当月新入职人数、当月离职人数
- [ ] **EMP-02**: 员工数据看板展示离职率（离职人数/(离职人数+期末人数)×100%）

### 组织架构可视化

- [ ] **EMP-03**: 管理员可查看组织架构可视化图表（部门名称→岗位名称→员工名称层级树）
- [ ] **EMP-04**: 组织架构支持按部门/岗位/员工检索定位

### 员工信息登记

- [x] **EMP-05**: 管理员可创建员工信息登记表（必填：姓名/部门/岗位/入职日期，可检索已有员工）
- [x] **EMP-06**: 管理员填写或转发员工填写：手机号码/住址/身份证正反面/银行卡正反面/学历证书/紧急联系人
- [x] **EMP-07**: 员工信息登记支持重新创建已入库员工的信息（以手机号或身份证号匹配，以最新版为准）
- [x] **EMP-08**: 员工信息登记提交后，数据更新到员工个人信息

### 办离职优化

- [ ] **EMP-09**: 离职待办列表展示事项/发起人/时间/状态/排序
- [ ] **EMP-10**: 管理员可审批离职申请（同意/驳回）
- [ ] **EMP-11**: 离职审批通过后可立即减员（跳转社保公积金减员）
- [ ] **EMP-12**: 减员完成后离职状态自动更新

### 花名册增强

- [ ] **EMP-13**: 花名册显示员工/状态/岗位薪资/在职年限/合同到期天数/手机号码
- [ ] **EMP-14**: 花名册点击"更多"跳转员工个人信息窗口（基本信息+员工档案：身份证/合同/银行卡/简历）
- [ ] **EMP-15**: 花名册支持按关键字搜索

### 下载表单

- [ ] **EMP-16**: 员工列表支持 Excel 格式下载导出
