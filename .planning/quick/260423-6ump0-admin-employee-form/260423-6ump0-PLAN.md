# Quick Plan: 管理员手动录入完整员工信息

## Task Description
在新增员工页面添加管理员手动输入所有员工信息的功能，让管理员可以一次性录入完整信息（包括薪资、紧急联系人等），而不是只录入基本信息和发送邀请。

## Implementation

### 任务 1: 扩展 Step 0 表单，增加完整信息录入

**Files:**
- `frontend/src/views/employee/EmployeeCreate.vue`

**Action:**
1. 在 Step 0 的表单中增加以下字段：
   - 试用期薪资（probation_salary）
   - 工资卡号（bank_card）
   - 紧急联系人姓名（emergency_contact）
   - 紧急联系人电话（emergency_phone）

2. 这些字段已在编辑模式中存在，需要在创建模式中也显示

3. 更新 form reactive 对象，确保新增字段被正确处理

**Verify:**
- npm run build 编译通过
- 创建模式下能看到所有新增字段

### 任务 2: 调整 Step 1 逻辑

**Files:**
- `frontend/src/views/employee/EmployeeCreate.vue`

**Action:**
1. Step 1 只保留推送方式选择
2. 移除已创建后的状态显示，统一在点击"完成"后创建员工并发送邀请

**Files:**
- `frontend/src/views/employee/EmployeeCreate.vue`
