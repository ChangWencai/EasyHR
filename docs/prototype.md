## 一、项目概述

### 1.1 产品简介

**EasyHR（易人事）** 是一款面向小微企业/个体户老板（10-50人规模）的轻量化、一站式人事管理APP。解决无专职HR的小老板在员工入职、社保、工资、个税、财务记账等基础人事事务中的操作痛点。

### 1.2 设计目标

- **简单、好用、省时间** — 老板打开APP第一时间知道要做什么
- **3步完成核心人事操作** — 无需专业知识
- **操作步骤 ≤ 3步** — 产品核心差异化

### 1.3 设计原则

| 原则 | 说明 |
|------|------|
| 简洁直观 | 核心功能一目了然，零学习成本 |
| 效率优先 | 数据概览 → 费用明细 → 操作入口，层级清晰 |
| 色彩引导 | 通过颜色区分操作优先级和状态类型 |
| 一致性 | 统一的组件规范和交互模式 |

## 七、状态定义

### 7.1 员工状态

| 状态 | 颜色 | 说明 |
|------|------|------|
| 在职 | `#10B981` (绿色) | 正常在岗 |
| 待增员 | `#8B5CF6` (紫色) | 社保公积金待办理 |
| 离职待办 | `#EF4444` (红色) | 离职流程待处理 |
| 已转出 | `#6B7280` (灰色) | 社保已转出 |
| 离职未转出 | `#EF4444` (红色) | 离职但社保未转出 |
| 离职未封存 | `#EF4444` (红色) | 离职但公积金未封存 |

### 7.2 缴费状态

| 状态 | 颜色 | 说明 |
|------|------|------|
| 正常 | `#10B981` (绿色) | 缴费正常 |
| 欠缴 | `#EF4444` (红色) | 存在欠缴 |
| 已转出 | `#6B7280` (灰色) | 已办理减员转出 |
| 待办 | `#8B5CF6` (紫色) | 待处理缴费 |

### 7.3 数据变化指标

| 指标 | 颜色 | 箭头 | 说明 |
|------|------|------|------|
| 增长 | `#10B981` (绿色) | ↑ | 相比上月增加 |
| 下降 | `#EF4444` (红色) | ↓ | 相比上月减少 |
